# ai-development

Praktische uitleg om AI-tools eenvoudig te integreren binnen GitHub en Visual Studio Code.

## Waarom AI integreren?

Met AI-tools kun je sneller:

- code schrijven en refactoren
- documentatie opstellen
- pull requests beoordelen
- tests en foutoplossingen voorbereiden

Zo wordt AI een vaste ondersteuning binnen je dagelijkse ontwikkelworkflow.

## AI in GitHub gebruiken

Een eenvoudige manier om AI binnen GitHub te gebruiken is via GitHub Copilot en gerelateerde GitHub-functies:

1. activeer GitHub Copilot voor je account of organisatie
2. open een repository op GitHub
3. gebruik AI bij issues, pull requests en code reviews
4. laat AI helpen met samenvattingen, verbetervoorstellen en documentatie

Hiermee voeg je AI toe zonder een aparte maatwerkoplossing te bouwen.

## AI in VS Code integreren

Binnen VS Code kun je AI snel toevoegen via extensies:

1. open **Extensions**
2. installeer een AI-extensie zoals **GitHub Copilot**
3. meld je aan met je GitHub-account
4. open je project en gebruik inline suggesties of chat

Veelgebruikte toepassingen in VS Code:

- code aanvullen tijdens het typen
- uitleg krijgen over bestaande code
- tests laten genereren
- commits of documentatie laten voorbereiden

## Praktische workflow

Een laagdrempelige aanpak is:

- gebruik GitHub voor samenwerking, reviews en issuebeheer
- gebruik VS Code voor dagelijkse ontwikkeling
- laat AI helpen bij repetitieve taken
- controleer altijd handmatig de output van AI

## Belangrijk aandachtspunt

AI-tools versnellen het werk, maar vervangen geen code review. Controleer daarom altijd:

- de juistheid van de voorgestelde code
- security en privacy van gegenereerde output
- aansluiting op de standaarden van je repository
